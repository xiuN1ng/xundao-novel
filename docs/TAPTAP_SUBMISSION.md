## TapTap 开发者中心操作步骤

### 第一步：注册/登录 TapTap 开发者账号

1. 访问 [TapTap 开发者中心](https://developer.taptap.io/)
2. 使用 TapTap 账号登录（如无账号则先注册）
3. 完成开发者实名认证（个人开发者需身份证认证）

### 第二步：创建游戏

1. 进入开发者中心，点击「创建游戏」
2. 选择游戏类型：**H5 小游戏**
3. 填写基本信息：
   - **游戏名称**：寻道·文字冒险
   - **游戏标签**：修仙 / 剧情 / 文字冒险
   - **支持语言**：简体中文

### 第三步：上传游戏包

1. 下载本目录下的 `xundao-novel.zip`
2. 在「版本管理」页面，上传 zip 包
3. 确认包体解压后包含 `index.html`（由平台自动识别入口）

### 第四步：填写游戏信息

| 字段 | 内容 |
|------|------|
| **游戏图标** | 1080×1080 PNG，建议使用封面图裁剪 |
| **游戏简介** | 一句话介绍：少年云泽的修仙问道之旅 |
| **详细介绍** | 详见下方「游戏介绍文案」 |
| **截图** | 竖屏截图 3-5 张（建议使用各章节关键场景图） |
| **运营商** | 个人 |
| **适龄提示** | 12+ |

### 第五步：提交审核

1. 确认所有必填项已填写
2. 点击「提交审核」
3. 审核周期：通常 3-7 个工作日

### 第六步：活动页面投稿

> 投稿地址（待定）：TapTap 开发者活动页
> 活动名称：TapTap H5 小游戏征集

1. 游戏审核通过后
2. 进入对应活动页面
3. 点击右下角【投稿】按钮
4. 填写游戏介绍（可参考下方「游戏介绍文案」）
5. 选择已发布的游戏版本
6. 发布投稿

## 游戏介绍文案（可直接复制）

### 一句话简介
少年云泽的修仙问道之旅，一场关于命运与抉择的沉浸式文字冒险。

### 详细介绍

**寻道·文字冒险**是一款中国水墨画风格的修仙剧情向文字冒险游戏。

**【故事简介】**
少年云泽在青云镇与爷爷相依为命，直到爷爷临终前道出一个惊天秘密——云泽体内封印着一头上古凶兽饕餮的魂魄。为查明真相、延续爷爷遗愿，云泽踏上了通往天道宗的修仙之路。然而，仙门之中暗流涌动，阴谋与背叛接踵而至……

**【游戏特色】**
- 🌙 沉浸式修仙剧情：9章完整故事线，多结局分支
- 🎨 中国水墨画风格：精心设计的场景插画与角色立绘
- ⚔️ 关键抉择影响剧情：你的选择将决定云泽的命运走向
- 🗺️ 多结局系统：6种不同结局等你探索

**【主要章节】**
- 序章：青云镇——爷爷的临终遗言
- 第1-3章：天道宗入门试炼
- 第4-6章：仙门磨砺与九宗大比
- 第7-9章：饕餮觉醒与最终对决

## 注意事项

1. **包体大小**：当前 3.5MB，远低于 TapTap 300MB 限制
2. **竖屏适配**：游戏设计为 9:16 竖屏，建议在竖屏设备/模拟器下测试

4. **TapTap SDK**：当前为平台桩，评分/分享功能由 TapTap 平台注入实现
5. **活动投稿**：建议在游戏审核通过后再进行投稿，审核期间可先准备介绍文案和截图
