# 寻道·文字冒险游戏 — 美术风格规范

> 版本：v1.0 | 更新日期：2026-04-05
> 目标平台：TapTap H5 | 分辨率：9:16 (1080×1920)

---

## 1. 整体美术风格定位

### 风格名称
**赛博水墨风（Ink-Punk）**
- 以中国传统水墨画为基底，融入现代游戏 UI 的简洁几何语言
- 区别于纯古风水墨：更锐利的线条、更高的信息密度、更适合 H5 小屏阅读
- 对标风格参考：《影之刃》《剑网3》水墨风格 + 现代扁平 UI

### 核心关键词
```
水墨渲染 · 留白意境 · 低饱和配色 · 几何 UI · 毛笔字体
```

### 禁止事项
- ❌ 日系赛璐璐厚涂风格
- ❌ 欧美写实 3A 渲染风格
- ❌ 韩式唯美磨皮风格
- ❌ 纯黑白水墨无彩色场景（仅限特殊情绪场景使用）

## 3. 插画风格规范

### 水墨渲染技法
- **远景**：泼墨晕染，层次模糊，浓淡五法（浓/淡/干/湿/焦）
- **中景**：勾勒为主，填充为辅，线条流畅有力
- **近景（人物）**：线条清晰，色彩饱和度略高于背景

### 光影规则
- **主光源**：顶部偏前（模拟日光顶光）
- **辅助光**：侧面轮廓光（仅用于重要角色）
- **点光源**：油灯/法阵/凶兽眼（制造氛围焦点）
- **禁止**：正面平光（无层次感）

### 构图规则

| 类型 | 构图 | 适用场景 |
|------|------|----------|
| 全屏背景 | 人物占 30% / 背景占 70% | 章节封面、氛围场景 |
| 人物立绘 | 人物占 60-70% / 背景占 30-40% | 角色登场、剧情对话 |
| 场景特写 | 无人物，纯场景 | 场景过渡、章节间 |
| 仰视构图 | 人物下方仰角 15-30° | 仙门大能、饕餮 |
| 俯视构图 | 人物上方俯角 15-30° | 战斗场面、群戏 |

| 卡片背景 | `#1E1E32` | 略浅玄黑 |
| 卡片边框 | `#3A3A5C` | 暗紫灰 |

| 次要文字 | `#8A8AAA` | 暗灰紫 |
| 强调色/按钮 | `#C9A96E` | 缃黄金 |
| 危险/战斗 | `#C94C4C` | 绛红 |
| 成功/治愈 | `#5A9E7C` | 青绿 |
| 仙法/特殊 | `#7B68B8` | 紫青 |

### UI 装饰元素
- **分隔线**：渐变消失线（玄黑→透明）
- **按钮**：圆角矩形（8px），玄黑底+缃黄边框
- **对话框**：毛笔笔触边缘效果（可用 CSS clip-path 模拟）
- **血条/灵力条**：水墨晕染风格，两端渐变消失

## 5. 字体规范

### 游戏内字体（优先顺序）
```
1. 系统默认：无（iOS/Android 各自系统字体）
2. Web Font：Noto Serif SC（Google Fonts）— 章节标题
3. Web Font：Noto Sans SC — 正文对话
4. Fallback：宋体 / 黑体 / sans-serif
```

### 字号规范（9:16 竖屏）

| 元素 | 字号 | 行高 |
|------|------|------|
| 章节标题 | 28-32px | 1.4 |

| 选项文字 | 20-22px | 1.6 |
| 小字/旁白 | 14-16px | 1.6 |

## 6. 分辨率与适配

### 基准分辨率
- **设计基准**：1080×1920（px）
- **输出格式**：PNG（场景/立绘）、WebP（UI 资源）

### 适配策略
- 场景插画：竖版 9:16，重要内容控制在安全区内（上下各留 15%边距）
- 立绘插画：人物中心占画面 60-70%，适配不同屏宽
- UI 资源：使用 9-patch 或 SVG 防止拉伸变形

### 性能预算（TapTap H5）
- 单张场景插画：≤ 500KB
- 角色立绘：≤ 300KB
- UI 资源总量：≤ 2MB
- 首屏加载：≤ 3s（可加 loading 过渡）

## 7. 动画风格（未来迭代）

> 当前版本为静态插画，以下规范为后续动画化预留

### 插画动画原则
- **慢速水墨晕染**：过渡动画 800-1200ms，ease-in-out
- **呼吸感**：立绘角色有极轻微上下浮动（3-5px，周期 3s）
- **粒子效果**：灵气/凶气用 Canvas 粒子叠加，不影响原图
- **禁止**：闪烁、突变、跳帧等快速动画（中国风水墨意境需要"静"）
